# ml-cup-2025-regression
Multi-output regression solution for the ML-CUP 2025 competition (ENNS). Stacking ensemble of Neural Networks (PyTorch), KNN, and SVR with Ridge meta-learner. Includes baseline comparisons (OLS, Lasso, Ridge), systematic hyperparameter tuning, early stopping, and 5-fold CV for out-of-fold meta-feature generation.
